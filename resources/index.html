About

Welcome to the Chip website!

This project provides a Coq specification and executable functions for change impact analysis, as used in build systems and tools for regression test selection. By extraction to OCaml code, practical certified tools for change impact analysis can be obtained as well.

This is an open source project, licensed under the MIT license.
